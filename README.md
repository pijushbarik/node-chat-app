# Practice Node Chat App

#### Install modules
```javascript
npm install
```
---

#### Build bundles (production)
```javascript
npm run build
```
---

#### Build bundles (dev)
```javascript
npm run build:dev
```
---

#### Start server
```javascript
npm start
```
---

#### Start development server
```javascript
npm run start:watch
```
---

#### Start test
```javascript
npm test
```
---

#### Start developement test server
```javascript
npm run test:watch
```
